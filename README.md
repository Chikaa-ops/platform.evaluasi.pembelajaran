## 🎯 Fitur Utama

### 1. 📝 **Soal & Kuis**
- Buat soal pilihan ganda, essay, dan matching
- Penilaian otomatis untuk pilihan ganda
- Bobot dan level kesulitan dapat disesuaikan
- Bank soal untuk reusability

### 2. 📐 **Rubrik Penilaian**
- Rubrik analitik dengan multiple kriteria
- Rubrik holistik untuk penilaian cepat
- Level pencapaian yang jelas (Excellent, Good, Fair, Poor)
- Cocok untuk tugas kompleks dan proyek

### 3. 📂 **Portofolio Pembelajaran**
- Dokumentasi perkembangan siswa sepanjang waktu
- Refleksi diri dan self-discovery
- Tracking progress dengan visual yang jelas
- Media komunikasi dengan orang tua

### 4. 👁️ **Observasi & Checklist**
- Pencatatan sistematis perilaku dan kehadiran
- Checklist yang dapat dikustomisasi
- Ideal untuk penilaian formatif dan monitoring
- Tracking partisipasi dan sikap siswa

### 5. 🧠 **Self-Assessment**
- Mendorong refleksi diri siswa
- Meningkatkan kesadaran diri (self-awareness)
- Format rating scale dan pertanyaan terbuka
- Membandingkan persepsi diri vs penilaian guru

### 6. 👥 **Peer-Assessment**
- Penilaian dari teman sebaya
- Pembelajaran kolaboratif yang lebih mendalam
- Feedback konstruktif antar siswa
- Mengembangkan keterampilan berpikir kritis

### 7. 📈 **Analytics & Dashboard**
- Dashboard statistik pembelajaran
- Identifikasi siswa yang membutuhkan intervensi
- Trend analysis per topik/materi
- Export laporan untuk reporting

## 🔧 Cara Menggunakan

### Quick Start (5 Menit)

1. **Buka platform** di browser
2. **Pilih tab fitur** yang ingin digunakan
3. **Buat instrumen evaluasi** (soal, rubrik, dll)
4. **Bagikan kepada siswa** (screenshot atau link)
5. **Monitor hasil** di analytics dashboard

### Workflow Lengkap

```
1. PERSIAPAN
   ↓ Tentukan TPK → Pilih metode evaluasi → Buat instrumen

2. IMPLEMENTASI
   ↓ Input ke platform → Siswa melakukan evaluasi → Sistem catat data

3. ANALISIS
   ↓ Lihat laporan → Analisis hasil → Identifikasi pola

4. TINDAK LANJUT
   ↓ Feedback → Remediasi/Pengayaan → Improve pembelajaran
```

## ❓ FAQ

### Q: Apakah platform ini gratis?
**A:** Ya! 100% gratis dan open source. Tidak ada biaya lisensi atau biaya tersembunyi.

### Q: Apakah data saya aman?
**A:** Sangat aman! Data tersimpan lokal di browser kamu, tidak dikirim ke server. Kamu punya full control atas data.

### Q: Bisa pakai offline?
**A:** Ya! Setelah dibuka sekali online, kamu bisa gunakan offline tanpa masalah.

### Q: Berapa siswa yang bisa digunakan?
**A:** Untuk versi ini, tidak ada batasan jumlah siswa. Semua tergantung kapasitas browser/komputer.

### Q: Gimana kalau data hilang?
**A:** Backup data secara berkala. Export hasil evaluasi atau simpan screenshot penting. Jangan clear browser cache tanpa backup.

### Q: Bisa customize warna/design?
**A:** Bisa! Edit file CSS di bagian `<style>` di file HTML. Semua kode tersedia dan bisa dimodifikasi.

### Q: Bisa digunakan di mobile?
**A:** Platform ini responsive dan bisa diakses di mobile, tapi UX tidak seoptimal desktop. Mobile app sedang dikembangkan.

### Q: Apakah perlu login?
**A:** Untuk versi basic ini tidak ada login. Multi-user dengan login akan ditambahkan di versi mendatang.

### Q: Bisa integrasi dengan Google Classroom?
**A:** Tidak untuk saat ini, tapi sedang dalam roadmap untuk versi mendatang.
